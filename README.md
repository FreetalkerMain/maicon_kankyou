## <div id="1">ライントレースの手引き</div>
### 手順
・systemworkbenchを入れる  
・cubemxをいれる https://www.st.com/ja/development-tools/stm32cubemx.html  
・stm32f303kマイコンでLちかをする  
・モータドライバを使ってモータを動かす  
・センサーの値を取得する  
・基盤を作る  
・走らせる
